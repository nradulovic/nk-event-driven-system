The following is the table containing abbreviations used in the API and source code:
  
  | Word                       | Abbreviation |
  | -------------------------- | ------------ |
  | interrupt                  | int          |
  | interrupt service routine  | isr          |
  | interrupt request          | irq          |
  | priority                   | prio         |
  | configuration              | config       |
  | invalid                    | invld        |
  | illegal                    | ilgl         |
  | attribute                  | attr         |
